# Реферальная программа (отдельный домен)

Отдельный сайт реферальной программы для развёртывания на домене, отличном от основного (например, blabla.su при основном сайте example.com).

## Быстрый старт

```bash
npm install
cp .env.example .env
# Отредактируйте .env
npm run dev
```

Откройте http://localhost:3002

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `NEXT_PUBLIC_REFERRAL_API_URL` | **Обязательно.** URL API основного сайта (например `https://api.example.com`) |
| `NEXT_PUBLIC_MAIN_SITE_URL` | URL основного сайта для ссылок «Пополнение / Вывод» (например `https://example.com`) |

## Настройка бэкенда

В `.env` основного сайта (backend) добавьте:

```env
REFERRAL_FRONTEND_ORIGIN=https://blabla.su
```

Чтобы CORS разрешал запросы с домена реферальной программы.

## Деплой

1. Укажите переменные окружения в настройках хостинга (Vercel, Netlify и т.д.)
2. Соберите: `npm run build`
3. Запустите: `npm start` или используйте статический экспорт

Порт по умолчанию: 3002 (чтобы не конфликтовать с основным frontend на 3000).
