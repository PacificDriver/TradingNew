# Шрифты для сайта

Чтобы подключённые шрифты **Scumbria** и **Aneliza** отображались, скачайте их и положите сюда:

| Шрифт   | Файлы (любой формат: .woff2, .woff, .ttf) | Ссылка для скачивания |
|---------|-------------------------------------------|------------------------|
| Scumbria | `Scumbria-Regular.woff2`, `Scumbria-Bold.woff2` | [Fontesk](https://fontesk.com/scumbria-font/) или [Online Fonts](https://online-fonts.com/fonts/scumbria/download) |
| Aneliza  | `Aneliza-Regular.woff2`, `Aneliza-SemiBold.woff2` или .ttf | [Шрифты Онлайн](https://fonts-online.ru/fonts/aneliza/download) |

После скачивания переименуйте файлы под имена выше или поправьте пути в `app/globals.css` в блоках `@font-face`.

Если файлов нет, сайт использует системный fallback (без потери вёрстки).
